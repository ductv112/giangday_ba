## Cách dùng

1. **Markdown gốc:** mở bằng VS Code / Obsidian / Typora — đọc dễ + edit nhanh
2. **Convert sang Word:** Pandoc `pandoc TPL-XXX.md -o TPL-XXX.docx` hoặc copy-paste vào Word
3. **Convert sang Excel:** template dạng table → copy-paste hoặc dùng Google Sheets import Markdown
4. **In ra giấy:** dùng cho workshop on-site, cho học viên điền tay
