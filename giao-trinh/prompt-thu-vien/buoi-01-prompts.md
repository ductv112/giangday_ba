# Buổi 1 — Prompt Library

## Prompt 1: Demo Claude Code đầu tiên — Sinh User Story

**Tool:** Claude Code (VS Code) hoặc ChatGPT
**Khi dùng:** Slide 13 Buổi 1 (demo live cho học viên thấy AI làm BA work nhanh)
**Mục tiêu:** Wow moment — AI sinh ra 5 user story chuẩn INVEST trong 30 giây

```
Tôi đang xây hệ thống đặt lịch sửa chữa điện tử – điện lạnh.
Stakeholder gồm: khách hàng, thợ kỹ thuật, điều phối viên admin.

Hãy phác hoạ 5 user story đầu tiên cho ROLE KHÁCH HÀNG,
theo format INVEST:
- As a [role], I want [feature], so that [benefit]
- Mỗi user story kèm 3 acceptance criteria dạng Given-When-Then
- Ngôn ngữ Việt
```

### Output mong đợi

5 user story format:
> *As a customer, I want to book repair appointment, so that...*
>
> Acceptance:
> 1. Given... When... Then...

### Tips dạy

- Show speed: 30 giây AI sinh ra cái BA cần 1 giờ viết tay
- Show quality: format chuẩn INVEST + Given-When-Then
- Cảnh báo: vẫn phải edit + verify, KHÔNG nộp thô
